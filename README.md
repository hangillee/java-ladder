- [온라인 코드 리뷰 과정](https://github.com/woowacourse/woowacourse-docs/blob/master/maincourse/README.md)

## 기능 요구사항

* 사람 이름 (Name)
    * 최대 5글자까지 부여할 수 있다.

* 사람 이름들 (Names)
    * 쉼표(,)를 기준으로 구분한다.
    * 중복을 허용하지 않는다.

* 사다리 (Lines)
    * 라인을 모아둔 일급 컬렉션.
    * 사용자가 입력한 사다리 높이 만큼 라인을 생성한다.

* 라인 (Line)
    * 라인 폭은 사람 이름을 기준으로 한다.
    * 가로 라인은 겹치지 않도록 해야 한다.

* 입력
    * 사람 이름
    * 사다리 높이

* 출력
    * 사다리 모양과 사람 이름을 함께 출력
